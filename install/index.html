esp32-rnd

Flash an ESP32-S3 board with one of these firmwares directly from your browser. No installation required.

Your browser doesn't support WebSerial. Use Chrome or Edge on desktop. Safari and Firefox don't work.

blink

Cycle the onboard RGB LED red → green → blue.

wifi-hello

Connect to your Wi-Fi and print the assigned IP over USB serial. Wi-Fi credentials are not embedded — open serial monitor (115200 baud) and follow the prompt, or rebuild from source with secrets.h filled in.

How this works

Your browser uses the WebSerial API to talk directly to the chip over USB, running a JavaScript port of esptool. Nothing gets installed on your computer.

Source
